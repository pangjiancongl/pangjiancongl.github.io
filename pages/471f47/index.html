flex布局

文档 (opens new window)

# 概念

Flex 是 Flexible Box 的缩写，意为”弹性布局”,用来为盒状模型提供最大的灵活性。

# 声明 Flex

.box {
  display: flex;
}

# 父容器属性

# 主轴方向

文档 (opens new window)

# 代码

.box {
  flex-direction: column | column-reverse | row | row-reverse;
}
1
2
3

# 参数

  • 主轴为水平方向
    • row（默认值）：自左而右排列
    • row-reverse：自右到左排列
  • 主轴为垂直方向
    • column：自上到下排列
    • column-reverse：自下到上排列

# 效果

和上面代码中属性顺序相对应

# 是否换行

文档 (opens new window)

# 代码

.box {
  flex-wrap: nowrap | wrap | wrap-reverse;
}
1
2
3

# 参数

  • nowrap 不换行
  • wrap 换行
  • wrap-reverse 换行,第一行在下面

# 效果

  • nowrap（默认）：不换行。
  • wrap：换行

  • wrap-reverse：换行，第一行在下方。

文档 (opens new window)

flex-flow 属性是 flex-direction 和 flex-wrap 的简写。

# 代码

.box {
  flex-flow: <flex-direction> || <flex-wrap>;

# 主轴上的对齐方式

文档 (opens new window)

# 代码

.box {
  justify-content: flex-start | flex-end | center | space-between | space-around;
}
1
2
3

# 参数

  • normal： 默认值 空
  • flex-start： 靠左排列
  • flex-end：靠右排列
  • center： 居中排列
  • space-between：两端靠边,中间元素居中，距离两边元素距离相等
  • space-around：每个元素都有一个相等左右外边距,并且相邻元素外边距不会合并

# 效果

# 交叉轴上的对齐方式

文档 (opens new window)

交叉轴:例如主轴是垂直，交叉轴就是水平

# 代码

.box {
  align-items: flex-start | flex-end | center | baseline | stretch;
}
1
2
3

# 参数

  • flex-start：交叉轴的起点对齐。
  • flex-end：交叉轴的终点对齐。
  • center：交叉轴的中点对齐。
  • baseline: 项目的第一行文字的基线对齐。
  • stretch：如果项目未设置高度或设为 auto，将占满整个容器的高度。

# 效果

# 多根轴线的对齐方式

文档 (opens new window)

该属性对单行弹性盒子模型无效。（即：带有 flex-wrap: nowrap）

# 代码

.box {
  align-content: flex-start | flex-end | center | space-between | space-around |
    stretch;
}
1
2
3
4

# 参数

  • start 所有行从容器的起始边缘开始填充。
  • end 所有行从容器的结束边缘开始填充。
  • flex-start：与交叉轴的起点对齐。
  • flex-end：与交叉轴的终点对齐。
  • center：与交叉轴的中点对齐。
  • space-between：与交叉轴两端对齐，轴线之间的间隔平均分布。
  • space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍。
  • stretch：轴线占满整个交叉轴。

# 排列顺序

文档 (opens new window)

# 简介

order 属性规定了弹性容器中的可伸缩项目在布局时的顺序。元素按照 order 属性的值的增序进行布局。拥有相同 order 属性值的元素按照它们在源代码中出现的顺序进行布局。

# 代码

/* integer 整数 */
.item {
  order: <integer>;
}
1
2
3
4

# 效果

图片上的数字为 order 值

# 放大

文档 (opens new window)

# 代码

/* 负数无效默认为0 */
.item {
  flex-grow: <number>;
}
1
2
3
4

# 效果

如果所有的兄弟项目都有相同的 flex-grow 系数，那么所有的项目将剩余空间（剩余空间是 flex 容器的大小减去所有 flex 项的大小加起来的大小）按相同比例分配，否则将根据不同的 flex-grow 定义的比例进行分配。

# 缩小

文档 (opens new window)

flex-shrink 属性指定了 flex 元素的收缩规则。flex 元素仅在默认宽度之和大于容器的时候才会发生收缩，其收缩的大小是依据 flex-shrink 的值。

# 代码

/* 负数无效默认为1 */
.item {
  flex-shrink: <number>;
}
1
2
3
4

# 效果

如果所有项目的 flex-shrink 属性都为 1，当空间不足时，都将等比例缩小。

# 元素在主轴方向上的初始大小

文档 (opens new window)

flex-basis 指定了 flex 元素在主轴方向上的初始大小。如果不使用 box-sizing 改变盒模型的话，那么这个属性就决定了 flex 元素的内容盒（content-box）的尺寸。

# 语法

flex-basis: < "width" > || content;
1

# 参数

  • auto : 默认值
  • content : 基于 flex 的元素的内容自动调整大小。
  • width ：不能为负，可以设置具体宽度，也可以设置百分比

# flex

flex-grow flex-shrink flex-basis 三个属性的简写

# 代码

后两个参数可选

.item {
  flex: none | [ < "flex-grow" > < "flex-shrink" >? || < "flex-basis" > ];
}
