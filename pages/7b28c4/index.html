    两数相加

    # 题目描述

    给你两个   非空 的链表，表示两个非负的整数。它们每位数字都是按照   逆序   的方式存储的，并且每个节点只能存储   一位   数字。

    请你将两个数相加，并以相同形式返回一个表示和的链表。

    你可以假设除了数字 0 之外，这两个数都不会以 0  开头。

    # 实例

    输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807.
    1
    2
    3

    # 思路

    1. 需要取出两个链表里面的对应每一位数字然后相加
    2. 如果链表长度不一样，另一个链表少的部分值要设置为 0
    3. 判断是否需要进位，如果需要进位就+1 不需要直接添加
    4. 判断链表是否为空，依次取出里面的值

    # 代码

    /**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} l1
 * @param {ListNode} l2
 * @return {ListNode}
 */
var addTwoNumbers = function (l1, l2) {
  // 进位---默认不需要进位
  let a = 0;
  let sum = new ListNode("0");
  let head = sum;
  // 只要l1或者l2 不为空就要继续循环 而且有进位也要继续循环
  while (a || l1 || l2) {
    // 获取 l1的值 如果为null 就设置为零 因为l2可能还有值还有继续相加
    let val1 = l1 != null ? l1.val : 0;
    // 获取l2的值
    let val2 = l2 != null ? l2.val : 0;
    // l1 + l2 的和 因为可能会有进位，所以要加上进位
    let r1 = val1 + val2 + a;
    // 判断是否需要进位
    a = r1 >= 10 ? 1 : 0;
    // 把值添加
    sum.next = new ListNode(r1 % 10);
    sum = sum.next;
    // 判断是否有值，有值就进行取值
    if (l1) {
      l1 = l1.next;
    }
    if (l2) {
      l2 = l2.next;
    }
  }
  return head.next;
};
