# 循环数组的几种方式

# for

性能最高

let arr = [1, 2, 3, 4];
for (let i = 0; i <= arr.length; i++) {
  // 循环体
}

# 改善 for

上面代码没循环一次就要获取一次 arr.length,造成了资源浪费

let arr = [1, 2, 3, 4];
let length = arr.length;
for (let i = 0; i <= length; i++) {
  // 循环体
}

# for in

运行速度慢 也可以用来循环对象

let arr = [1, 2, 3, 4];
for (index in arr) {
  // 循环体
  console.log(index); // 索引
}

# for of

运行速度较慢但高于 for in 可以用于遍历可迭代的对象，例如 String、Array、Map 和 Set 等

const arr = [3, 5, 7];
const str = "hello";
for (let i of arr) {
  console.log(i); //  3, 5, 7
}
for (let i of str) {
  console.log(i); //  'h', 'e', 'l', 'l', 'o'
}
