常见面试题

# css 标准盒盒模型和怪异盒模型

css 中的 Boxmodel 分两种一种是 W3C 标准一种是 IE 采用的标准

# 标准模式下

宽度 = width + margin(左右) + padding(左右) + border(左右)

# 怪异模式下

宽度 = width( width + padding(左右) + border(左右)) + margin(左右)

# box-sizing

# 语法

box-sizing ： content-box || border-box || inherit;
1

  • content-box 采用标准模式解析计算，也是默认模式；

  • border-box 采用怪异模式解析计算；

  • inherit 规定应从父元素继承 box-sizing 属性的值。

# sessionStorage 、localStorage 和 cookie 之间的区别

# 相同点

cookie，localStorage，sessionStorage 都是在客户端保存数据的

# 不同点

# 大小

cookie 只有 4k localStorage，sessionStorage 相对存储空间比较大有 5M 左右

# 生命周期

cookie: 如果设置有效期，根据有效期决定，如果没有设置浏览器关闭失效 localStorage: 永久有效，关闭浏览器不会消失，除非主动删除 sessionStorage: 仅在当前会话下有效，关闭浏览器会销毁

# 使用流量

cookie: 数据会在 HTTP 中携带在浏览器和服务器间来回传递

sessionStorage 和 localStorage:仅在客户端（即浏览器）中保存，不参与和服务器的通信

# 作用域

sessionStorage 不在不同的浏览器窗口中共享，即使是同一个页面； localStorage 和 cookie 在所有同源窗口是共享的

# cookie、session 区别

  • cookie 存储于浏览器端，而 session 存储于服务端

  • cookie 不是很安全，别人可以分析存放在本地的 COOKIE 并进行 COOKIE 欺骗，考虑到安全应当使用 session；

  • session 会在一定时间内保存在服务器上。当访问增多，会比较占用你服务器的性能。考虑到减轻服务器性能方面，应当使用 COOKIE；
